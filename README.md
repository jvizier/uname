# uname

## uname for centralized env


* **Overview** :
  - A Shell script to determine os release in variable way.
  - The aim is to build centralized env path (via nfs etc) or to be used in scripts / Mirrors

* **Example Usage** :

    `/Linux/Ubuntu/raring/x86_64`

* **Comparable Tools** :
  - lsb_release
  - original uname
